## 🍽️ Nossos Produtos

### [Tableflow](#) — Gestão de mesas e pedidos

Plataforma de gerenciamento de pedidos via QR Code para restaurantes. Clientes acessam o cardápio digital, fazem pedidos e chamam o garçom diretamente pela mesa — sem app, sem fila, sem atrito.

**Principais funcionalidades:**
- Cardápio digital dinâmico por QR Code
- Pedidos em tempo real com notificações para a cozinha
- Gestão de mesas e controle de comanda
- Painel administrativo para o estabelecimento
- Impressão térmica via ESC/POS (USB e Ethernet)

**Stack:** Node.js · Next.js · MongoDB · AWS S3 · Railway · Vercel

---

### [Servin](#) — Hospitalidade além do restaurante

Extensão do modelo Tableflow para hotéis, motéis e pousadas. Adaptado para fluxos de quarto — solicitações de serviço, cardápio de room service e comunicação interna — com uma interface de luxo pensada para o hóspede.

**Principais funcionalidades:**
- QR Code por quarto com identidade visual personalizada
- Fluxo de pedidos e solicitações adaptado para hospedagem
- Dark UI com estética premium
- Backend e autenticação independentes por unidade

**Stack:** Node.js · Next.js · MongoDB · AWS SES · Railway · Vercel
